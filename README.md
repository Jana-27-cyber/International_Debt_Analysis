# 🌍 International Debt Analysis System

## 📌 Project Overview

The International Debt Analysis System is a data analytics project developed using **Python, SQLite, Streamlit, and SQL**.

The project analyzes international debt data from multiple countries and indicators, providing insights through SQL analytical queries and interactive dashboards.

---

## 🎯 Objectives

* Clean and preprocess debt datasets
* Handle missing values using Forward Fill and Backward Fill
* Transform data from Wide Format to Long Format
* Merge metadata files
* Store cleaned data in SQLite database
* Perform SQL analytical queries
* Build an interactive Streamlit dashboard

---

## 📂 Dataset Files

The project uses the following datasets:

1. IDS_ALLCountries_Data.csv
2. IDS_CountryMetaData.csv
3. IDS_SeriesMetaData.csv
4. IDS_FootNoteMetaData.csv
5. Country-Series - Metadata.csv
